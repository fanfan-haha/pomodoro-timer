# Pomodoro Timer

一个基于 Electron 的桌面番茄钟应用，Observatory Dark 设计风格，支持专注计时、休息提醒和番茄数量统计。

## 功能

- **三种模式**：专注 25 分钟 / 短休息 5 分钟 / 长休息 15 分钟
- **辉光进度环**：SVG 发光滤镜 + 环境径向光晕，颜色随模式切换
- **自动切换**：专注结束自动进入休息，每完成 4 个番茄后触发长休息
- **系统通知**：每次切换模式时弹出 macOS 原生通知
- **数据统计**：今日番茄数、本轮计数、历史总计，本地持久化存储

## 设计风格

| 模式 | 强调色 | 说明 |
|:---:|:---:|:---|
| FOCUS | 琥珀黄 `#e8a020` | 专注 25 分钟 |
| SHORT | 翡翠绿 `#20d090` | 短休息 5 分钟 |
| LONG  | 天蓝色 `#20b8e8` | 长休息 15 分钟 |

- 字体：**Orbitron**（计时器）+ **Barlow Condensed**（界面文字）
- 背景：深空黑 `#070710` 配细点阵网格
- 进度环：SVG `feGaussianBlur` 辉光效果

## 快速开始

**环境要求**：Node.js 18+

```bash
npm install
npm start
```

## 项目结构

```
├── main.js       # Electron 主进程（窗口管理、系统通知）
├── renderer.js   # 渲染进程（计时逻辑、统计存储）
├── index.html    # 界面结构
├── styles.css    # 样式（Observatory Dark 主题）
└── package.json
```

## 番茄工作法

1. 选择一项任务，点击 **START**
2. 专注工作 25 分钟，不受打扰
3. 休息 5 分钟
4. 每完成 4 个番茄，享受 15 分钟长休息
